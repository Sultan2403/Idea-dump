# Idea Dump

## Project Overview
This project serves as a centralized location for users to dump their ideas and enhance their creative processes.

## Features
- User authentication
- Adding, editing, and deleting ideas
- Sorting and categorizing ideas
- Search functionality
- User-friendly interface

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB
- Authentication: JWT
- Deployment: Heroku

## Fun Note
This is the first fullstack app that's actively being improved, and I'm open to suggestions! Feel free to contribute your ideas and help make this project better!
