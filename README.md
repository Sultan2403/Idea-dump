# 🚀 Idea Dump

**My first fullstack project, and I'm actively improving it!** 💪

Idea Dump is a simple web app for dumping and managing your ideas. Think of it like a personal notes app, but specifically designed for capturing and organizing those brilliant thoughts before they slip away.

## 🎯 Features

- ✅ **Create & Organize Ideas** - Quickly jot down your ideas with a clean interface
- ✅ **Edit & Delete** - Modify or remove ideas anytime
- ✅ **View Ideas** - Browse through all your ideas in an organized manner
- ✅ **User Authentication** - Secure login and registration
- 🔄 **Voice Recording** - *In Progress* (coming soon!)

## 🌐 Try It Out

**Live Demo:** [https://idea-dump-pro.vercel.app](https://idea-dump-pro.vercel.app)

## 🛠️ Tech Stack

### Frontend
- **React 19** - UI library
- **Vite** - Build tool
- **React Router** - Navigation
- **Material-UI** - UI components
- **Tailwind CSS** - Styling
- **Axios** - HTTP client

### Backend
- **Node.js** - Runtime
- **Express** - Web framework
- **MongoDB** - Database
- **Mongoose** - MongoDB ODM
- **JWT** - Authentication
- **bcryptjs** - Password hashing
- **Celebrate** - Data validation
